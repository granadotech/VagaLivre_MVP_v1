# ⚠️ VagaLivre - MVP v1 (VERSÃO CONGELADA)

Esta versão representa o **MVP funcional completo do VagaLivre**.

## 🚫 Importante
Esta versão está **congelada** e **não deve ser alterada**.

## 🎯 Finalidade desta versão
- Demonstrações do produto
- Apresentações para usuários e parceiros
- Validação de conceito (MVP)

## 📦 O que está implementado
- Cadastro de vagas
- Disponibilização de vaga
- Busca de vagas disponíveis
- Reserva de vaga
- Cancelamento de disponibilidade
- Cancelamento de reserva (com regra de 1h)
- Controle de status (disponível, reservada, cancelada, finalizada)
- Histórico inteligente (Ativas vs Passadas)
- Identificação de quem cancelou a reserva

## 🚀 Evolução do produto
Toda evolução do sistema deve ser feita na versão:

👉 **VagaLivre_MVP_v2**

---

## 🧠 Observação
Esta versão deve permanecer estável como referência de funcionamento do produto.

Qualquer melhoria, ajuste ou nova funcionalidade deve ser feita na V2.
